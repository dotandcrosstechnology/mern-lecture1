ES6+ Features Recap

Let and Const

let and const provide block-scoped variables, solving many issues with var.

Variable Declarations 
// var vs let vs const
var x = 1;
let y = 2;
const z = 3;

// Block scoping demonstration
if (true) {
    var varVariable = "I'm function scoped";
    let letVariable = "I'm block scoped";
    const constVariable = "I'm also block scoped";
}

console.log(varVariable); // Works
// console.log(letVariable); // ReferenceError
// console.log(constVariable); // ReferenceError

Arrow Functions

Arrow functions provide a concise syntax and lexical this binding.

Arrow Functions Examples 
// Traditional function
function traditional(a, b) {
    return a + b;
}

// Arrow function
const arrow = (a, b) => a + b;

// Arrow function with single parameter
const square = x => x * x;

// Arrow function with multiple lines
const complexArrow = (name, age) => {
    const message = `Hello, I'm ${name} and I'm ${age} years old`;
    return message.toUpperCase();
};

Destructuring

Destructuring allows unpacking values from arrays or properties from objects into distinct variables.

Destructuring Examples 
// Array destructuring
const numbers = [1, 2, 3, 4, 5];
const [first, second, ...rest] = numbers;

// Object destructuring
const person = {
    name: 'John Doe',
    age: 30,
    city: 'New York',
    country: 'USA'
};

const { name, age, ...location } = person;

// Nested destructuring
const user = {
    id: 1,
    profile: {
        username: 'johndoe',
        social: {
            twitter: '@johndoe'
        }
    }
};

const { profile: { username, social: { twitter } } } = user;

Template Literals and String Methods

Template literals provide an easy way to interpolate variables and expressions into strings.

Template Literals 
const name = 'Anas';
const company = 'Dot and Cross Technology';
const age = 25;

// Basic template literal
const greeting = `Hello, my name is ${name}!`;

// Multi-line strings
const multiLine = `
    This is a multi-line string.
    It preserves line breaks.
    Very useful for HTML templates.
`;

// Expression evaluation
const calculation = `The result is: ${5 + 3 * 2}`;

// Function calls in template literals
const formatName = name => name.toUpperCase();
const formatted = `Hello, ${formatName(name)}!`;

// Tagged template literals
function highlight(strings, ...values) {
    return strings.reduce((result, string, i) => {
        return result + string + (values[i] ? `${values[i]}` : '');
    }, '');
}

const highlighted = highlight`My name is ${name} and I work at ${company}`;

Spread/Rest Operators

The spread (...) operator allows an iterable to expand in places where 0+ arguments are expected.

Spread Operator Examples 
// Array spreading
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const combined = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6]

// Object spreading
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };
const merged = { ...obj1, ...obj2 }; // { a: 1, b: 2, c: 3, d: 4 }

// Function arguments
const numbers = [1, 2, 3, 4, 5];
Math.max(...numbers); // 5

// Copying arrays/objects
const originalArray = [1, 2, 3];
const copiedArray = [...originalArray];

const originalObj = { name: 'John', age: 30 };
const copiedObj = { ...originalObj };

Rest Parameters

Rest Parameters 
// Rest parameters in functions
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

sum(1, 2, 3, 4, 5); // 15

// Mixed parameters
function greet(greeting, ...names) {
    return `${greeting} ${names.join(', ')}!`;
}

greet('Hello', 'Alice', 'Bob', 'Charlie');

// Rest in destructuring
const [first, second, ...others] = [1, 2, 3, 4, 5];
const { name, ...rest } = { name: 'John', age: 30, city: 'NYC' };

Modules (import/export)

ES6 modules allow you to organize code into reusable pieces with explicit dependencies.

math.js - Module Example 
// Named exports
export const PI = 3.14159;

export function add(a, b) {
    return a + b;
}

export function multiply(a, b) {
    return a * b;
}

// Default export
export default function subtract(a, b) {
    return a - b;
}

// Export object
const calculator = {
    divide(a, b) {
        return b !== 0 ? a / b : 'Cannot divide by zero';
    },
    power(base, exponent) {
        return Math.pow(base, exponent);
    }
};

export { calculator };
main.js - Import Examples 
// Named imports
import { add, multiply, PI } from './math.js';

// Default import
import subtract from './math.js';

// Import all as namespace
import * as MathUtils from './math.js';

// Mixed imports
import subtract, { add, calculator } from './math.js';

// Rename imports
import { add as sum, multiply as product } from './math.js';

// Dynamic imports (ES2020)
const loadMath = async () => {
    const mathModule = await import('./math.js');
    return mathModule;
};

Array Methods (map, filter, reduce, forEach)

Modern array methods provide powerful functional programming capabilities for data manipulation.

Map Method

Reduce Method

Array.reduce() Examples 
// Basic reduce - sum of numbers
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((acc, num) => acc + num, 0);
const product = numbers.reduce((acc, num) => acc * num, 1);

// Finding maximum value
const max = numbers.reduce((max, num) => num > max ? num : max);

// Grouping objects
const people = [
    { name: 'Alice', department: 'Engineering' },
    { name: 'Bob', department: 'Marketing' },
    { name: 'Charlie', department: 'Engineering' },
    { name: 'David', department: 'Marketing' }
];

const groupedByDepartment = people.reduce((acc, person) => {
    if (!acc[person.department]) {
        acc[person.department] = [];
    }
    acc[person.department].push(person.name);
    return acc;
}, {});

// Counting occurrences
const words = ['apple', 'banana', 'apple', 'cherry', 'banana', 'apple'];
const wordCount = words.reduce((acc, word) => {
    acc[word] = (acc[word] || 0) + 1;
    return acc;
}, {});

forEach and Other Methods

forEach and Additional Array Methods 
// forEach - for side effects
const names = ['Alice', 'Bob', 'Charlie'];
names.forEach((name, index) => {
    console.log(`${index + 1}. ${name}`);
});

// find and findIndex
const users = [
    { id: 1, name: 'Alice', active: true },
    { id: 2, name: 'Bob', active: false },
    { id: 3, name: 'Charlie', active: true }
];

const activeUser = users.find(user => user.active);
const bobIndex = users.findIndex(user => user.name === 'Bob');

// some and every
const numbers = [2, 4, 6, 8];
const hasEven = numbers.some(num => num % 2 === 0); // true
const allEven = numbers.every(num => num % 2 === 0); // true

// includes
const fruits = ['apple', 'banana', 'orange'];
const hasApple = fruits.includes('apple'); // true

Method Chaining

Chaining Array Methods 
// Complex data transformation with method chaining
const sales = [
    { product: 'Laptop', amount: 1200, category: 'Electronics' },
    { product: 'Phone', amount: 800, category: 'Electronics' },
    { product: 'Book', amount: 25, category: 'Education' },
    { product: 'Tablet', amount: 400, category: 'Electronics' },
    { product: 'Course', amount: 150, category: 'Education' }
];

const highValueElectronics = sales
    .filter(sale => sale.category === 'Electronics')
    .filter(sale => sale.amount > 500)
    .map(sale => ({
        ...sale,
        discountedPrice: sale.amount * 0.9
    }))
    .reduce((total, sale) => total + sale.discountedPrice, 0);

console.log(`Total high-value electronics with discount: ${highValueElectronics}`);
